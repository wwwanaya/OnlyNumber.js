OnlyNumber.js
=============

This is a small pluggin for make a HTML input with only numbers allowed on that input.

Easy to use
===========
Syntax:
onlyNumber($('#ID_Input'), $('#ID_Input'), $(ID_Input), ..);

Hope you understand is very easy.
